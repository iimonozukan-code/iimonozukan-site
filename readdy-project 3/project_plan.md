# いいもの図鑑

## 1. プロジェクト概要
Instagram・YouTube・TikTokで紹介してきたガジェットや生活雑貨を一覧できる図鑑型カタログサイト。各商品にAmazon・楽天・Yahoo・AliExpressの購入リンクを付与し、検索・絞り込み・並び替えで快適に商品を探せる。

- ターゲット：SNSで商品情報を発信しているインフルエンサー/クリエイターとそのフォロワー
- コアバリュー：紹介商品を図鑑のように美しく整理し、気になる商品にすぐたどり着ける

## 2. ページ構成
- `/` - ホーム（商品一覧 + 検索・絞り込み）

## 3. コア機能
- [x] 検索ボックス：商品名のリアルタイム絞り込み
- [x] カテゴリ絞り込み：機械もの / 生活もの / 情報もの
- [x] モール絞り込み：Amazon / 楽天 / Yahoo / AliExpress
- [x] 並び替え：紹介日の新しい順
- [x] 商品カードグリッド：画像・商品名・紹介日・購入ボタン
- [x] 購入ボタン：該当モールのみ表示、新規タブで開く
- [x] 多言語対応：日本語 / 英語切替（react-i18next）
- [x] AIカスタマーチャット：Readdy Agent Widget（右下フローティングボタン）

## 4. データモデル設計
データベース不要。`src/mocks/products.ts` にJSON配列で一元管理。

```typescript
interface Product {
  name: string;
  date: string;
  category: '機械もの' | '生活もの' | '情報もの';
  image: string;
  links: {
    amazon: string | null;
    rakuten: string | null;
    yahoo: string | null;
    aliexpress: string | null;
  };
}
```

## 5. バックエンド / サードパーティ連携計画
- Supabase: 不要（静的データのため）
- Shopify: 不要
- Stripe: 不要
- 外部リンク: Amazon, 楽天, Yahoo, AliExpress の商品ページURL
