# 新商品の追加方法

新商品は「画像」と「新タグ入りリンク」を自動で付けられます。2つのやり方があります。

---

## 方法A：いちばんかんたん（Coworkで頼む）

チャットでこう送るだけ：

> 新商品これ追加して
> Amazon: （商品ページのURL）
> 楽天: （URL）  ※あれば
> Yahoo: （URL）  ※あれば
> AliExpress: （URL）  ※あれば
> カテゴリ: 機械もの（など）

→ こちらでAmazonの公式画像を取得し、新タグ（`iimonozukan-media-22`）を付けて `products.ts` に追加します。**今日からできます。**

---

## 方法B：自分でPA-API自動取得（まとめて入れたい時向け）

> ⚠️ PA-APIはアクセス活性化に**最大48時間**、かつ過去30日で対象売上10件以上が条件。活性化後に使えます。

1. プロジェクト直下に `.env` を作る（`.env.example` をコピー）。Amazonのキーを記入：
   ```
   PAAPI_ACCESS_KEY=（アクセスキー）
   PAAPI_SECRET_KEY=（シークレットキー）
   PAAPI_PARTNER_TAG=iimonozukan-media-22
   ```
   ※ `.env` はGitに上げない設定済み（公開されません）。

2. `src/mocks/products.ts` に新商品を追加。`image` は空のまま、`asin` を入れる：
   ```ts
   {
     "name": "商品名",
     "date": "2026-06-20",
     "category": "機械もの",
     "image": "",
     "asin": "B0XXXXXXXX",
     "links": { "amazon": null, "rakuten": null, "yahoo": null, "aliexpress": null }
   }
   ```

3. ターミナルで：
   ```
   npm install        # 初回のみ
   npm run enrich:amazon
   ```

4. `asin` があって `image` が空の商品に、公式画像とAmazonリンク（新タグ入り）が自動で入ります。

### ASIN（商品ID）の見つけ方
Amazon商品ページURLの `/dp/` の後ろにある **10桁**（例：`amazon.co.jp/dp/B0FZP3CDP8` → `B0FZP3CDP8`）。

---

## メモ
- 既存159件は今のまま（画像はLinktree表示のまま）。この自動化は**今後の新商品向け**です。
- 楽天／Yahoo／AliExpressのリンクは手動で `links` に貼ってください（自動化はAmazon画像＋Amazonリンクのみ）。
